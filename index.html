Pasos para abrir Github

  1. Identificar el navegador en la computadora (chrome, firefox, edge, etc.)

  2. Identificar la barra de direcciones en el navegador

  3. Teclear la dirección del github "github.com/yuutamk"

  4. identificar el repositorio del curso, puede estar en la parte inferior al mostrar mas actividades o bien podemos buscar en el buscador de la pagina.

  5. Damos clic al repositorio del curso

  6. en el menú identificamos el tema que corresponde

Pasos para abrir el IDE de C++

  1. Abrimos el menu de inicio e identificamos el programa en cuestión, puede encontrarse en el menu de todos los programas o bien comenzamos a teclear el nombre del programa que es "dev-C++"

  2. Damos clic al ejecutable del programa

  3. Úbicamos el menú file, submenu new,Source File

  4. Guardamos el archivo creado con Ctrl + S o bien en el menu File, Save As

  5. Úbicamos la ruta de nuestra carpeta ya creada y lo guardamos con cualquier nombre, importante que termine con la extensión ".cpp" (punto cpp)

  6. Programamos nuestro codigo y para compilar úbicamos el menu "excecute" despues damos en "compile & run" o con la tecla de funcion "F11"
